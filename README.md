# HumanDesign

Учебное веб-приложение для построения карты Ба-цзы на основе даты и времени рождения пользователя.  
Проект разработан в учебных целях для практики создания веб-приложений на Django, работы с базой данных PostgreSQL и контейнеризации приложения с помощью Docker.

## О проекте

HumanDesign — это веб-приложение, которое позволяет сформировать карту Ба-цзы на основе введённых пользователем данных.  

Проект был создан как один из первых полноценных backend-проектов для практики разработки серверной логики, работы с базой данных и настройки окружения для запуска веб-приложения.

Приложение принимает данные пользователя (дату и время рождения), обрабатывает их на сервере и формирует карту Ба-цзы с использованием специализированной библиотеки.

## Стек технологий

Backend:

## Функциональность

- ввод данных пользователя (дата и время рождения)
- обработка данных на стороне сервера
- генерация карты Ба-цзы
- сохранение данных в базе PostgreSQL
- запуск приложения в контейнерах Docker
- управление несколькими сервисами через Docker Compose

## Запуск проекта

### Запуск через Docker

1. Клонировать репозиторий

`git clone https://github.com/KiRilLTatar/HumanDesign.git`

2. Перейти в папку проекта

`cd HumanDesign`

3. Запустить контейнеры

`docker compose up --build`

После запуска приложение будет доступно по адресу:

`http://localhost:8080`

### Запуск без Docker

1. Установить зависимости

`pip install -r requirements.txt`

2. Применить миграции

`python manage.py migrate`

3. Запустить сервер разработки

`python manage.py runserver`
