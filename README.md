# HumanDesign

Учебное веб-приложение для построения карты Ба-цзы на основе даты и времени рождения пользователя.  
Проект разработан в учебных целях для практики создания веб-приложений на Django, работы с базой данных PostgreSQL и контейнеризации приложения с помощью Docker.

## О проекте

HumanDesign — учебное веб-приложение для построения карты Ба-цзы на основе даты и времени рождения пользователя.

Проект разработан на фреймворке **Django** с использованием архитектурного подхода **SSR (Server-Side Rendering)**. Вся основная логика обработки данных выполняется на стороне сервера: пользователь вводит исходные данные через веб-интерфейс, после чего сервер обрабатывает их, формирует карту Ба-цзы и возвращает готовую страницу с результатами.

В рамках проекта была реализована backend-логика приложения, работа с базой данных PostgreSQL и контейнеризация окружения с использованием Docker. Приложение разворачивается в контейнеризированной среде через Docker Compose, что позволяет запускать сервисы приложения и базы данных как единое окружение.

Проект был создан в учебных целях как один из первых полноценных веб-проектов для практики:

- разработки backend-логики на Django
- работы с архитектурой SSR-приложений
- интеграции веб-приложения с PostgreSQL
- контейнеризации веб-сервисов с помощью Docker
- запуска многосервисного окружения через Docker Compose

## Запуск проекта

### Запуск через Docker

1. Клонировать репозиторий

`git clone https://github.com/KiRilLTatar/HumanDesign.git`

2. Перейти в папку проекта

`cd HumanDesign`

3. Запустить контейнеры

`docker compose up --build`

После запуска приложение будет доступно по адресу:

`http://localhost:8080`

### Запуск без Docker

1. Установить зависимости

`pip install -r requirements.txt`

2. Применить миграции

`python manage.py migrate`

3. Запустить сервер разработки

`python manage.py runserver`
